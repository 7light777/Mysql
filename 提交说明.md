# 篮球数据管理系统提交说明

本目录为课程工程提交版，已从原工程中剔除编译产物、IDE 配置、Git 仓库、本地 Maven 仓库和历史临时目录。

## 提交目录

提交时建议压缩整个目录：

```text
basketball-management-system-submit
```

## 目录内容

```text
basketball-management-system-submit/
├── pom.xml
├── README.md
├── run-local.ps1
├── docs/
│   └── api-test.md
└── src/
    └── main/
        ├── java/
        └── resources/
```

## 核心文件说明

- `pom.xml`：Maven 项目配置，Java 17，Spring Boot 3.3.x，MyBatis，Thymeleaf，MySQL。
- `src/main/java`：后端代码，采用 Controller、Service、Mapper 分层结构。
- `src/main/resources/mapper`：MyBatis XML SQL 文件。
- `src/main/resources/db`：数据库建表、演示数据、触发器、存储过程、视图脚本。
- `src/main/resources/templates`：Thymeleaf 页面模板。
- `src/main/resources/static`：CSS、JavaScript、球员头像、球队 Logo 等静态资源。
- `docs/api-test.md`：接口测试和演示说明。

## 数据库脚本执行顺序

```text
1. src/main/resources/db/01_schema.sql
2. src/main/resources/db/02_demo_data.sql
3. src/main/resources/db/04_position_display_patch.sql
4. src/main/resources/db/05_fix_scout_notes.sql
5. src/main/resources/db/03_trigger_procedure_view.sql
```

说明：`03_trigger_procedure_view.sql` 中包含触发器、存储过程和视图，建议在数据导入完成后执行。

## 运行方式

先修改：

```text
src/main/resources/application.yml
```

将数据库用户名和密码改为本机 MySQL 配置，然后执行：

```bash
mvn spring-boot:run
```

或打包运行：

```bash
mvn clean package -DskipTests
java -jar target-build/basketball-management-system-0.0.1-SNAPSHOT-boot.jar
```
